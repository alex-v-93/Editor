========================================================================
    БИБЛИОТЕКА АКТИВНЫХ ШАБЛОНОВ. Обзор проекта SimpleEditorHandlers
========================================================================

Этот проект SimpleEditorHandlers автоматически создан мастером приложений для использования в качестве основы при написании собственной библиотеки динамической компоновки (DLL).

В этом файле представлена сводка содержимого всех файлов, входящих в состав проекта.

SimpleEditorHandlers.vcxproj
    Это основной файл проекта VC++, создаваемый с помощью мастера приложений. Он содержит данные о версии языка Visual C++, использованной для создания файла, а также сведения о платформах, конфигурациях и функциях проекта, выбранных с помощью мастера приложений.

SimpleEditorHandlers.vcxproj.filters
    Это файл фильтров для проектов VC++, созданный с помощью мастера приложений. Он содержит сведения о сопоставлениях между файлами в вашем проекте и фильтрами. Эти сопоставления используются в среде IDE для группировки файлов с одинаковыми расширениями в одном узле (например CPP-файлы сопоставляются с фильтром "Исходные файлы").

SimpleEditorHandlers.idl
    В этом файле содержатся определения библиотеки типов, интерфейсов и компонентных классов, определенных в проекте на языке IDL.
    Этот файл будет обработан компилятором MIDL для создания следующих компонентов.
        Определения интерфейса C++ и объявления идентификаторов GUID (SimpleEditorHandlers.h).
        Определения идентификаторов GUID                                (SimpleEditorHandlers_i.c).
        Библиотека типов                                  (SimpleEditorHandlers.tlb).
        Код маршалинга                                 (SimpleEditorHandlers_p.c и dlldata.c).

SimpleEditorHandlers.h
    В этом файле содержатся определения интерфейса C++ и объявления идентификаторов GUID для элементов, определенных в файле SimpleEditorHandlers.idl. В процессе компиляции файл повторно создается компилятором MIDL.

SimpleEditorHandlers.cpp
    В этом файле содержится карта объектов и реализация компонентов экспорта библиотек DLL.

SimpleEditorHandlers.rc
    Он представляет собой список всех ресурсов Microsoft Windows, используемых программой.

SimpleEditorHandlers.def
    Это файл определений модуля, который предоставляет компоновщику сведения о компонентах экспорта, необходимых для библиотеки DLL. В файле содержатся следующие компоненты экспорта:
        DllGetClassObject
        DllCanUnloadNow
        DllRegisterServer
        DllUnregisterServer
        DllInstall

/////////////////////////////////////////////////////////////////////////////
Другие стандартные файлы:

StdAfx.h, StdAfx.cpp
    Эти файлы используются для построения файла предкомпилированного заголовка (PCH) с именем SimpleEditorHandlers.pch и файла предкомпилированных типов с именем StdAfx.obj.

Resource.h
    Стандартный файл заголовков, в котором определяются идентификаторы ресурсов.
